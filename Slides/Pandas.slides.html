Pandas - an Introduction¶

Easy-to-use Data Structures and Data Analysis Tool¶

Overview¶

  • Labeled array data structures (Series and DataFrame)
  • Simple axis label-based indexing and multi-level / hierarchical axis indexing
  • Integrated engine for aggregating and transforming data sets, e.g. groupby method
  • Timestamp generation and custom date offsets enabling the implementation of customized frequencies
  • Input/Output tools: loading tabular data from all sort of file formats (CSV, delimited, Excel 2003, etc.)
  • Comprehensive statistics toolbox (rolling mean, rolling standard deviation, etc.)

Source of this introduction: Pandas Documentation and Enthought Cheat Sheet.

Data structures¶

Dimensions Name Description
1 Series 1D labeled homogeneously-typed array
2 DataFrame General 2D labeled, size-mutable tabular structure with potentially heterogeneously-typed columns
3 Panel General 3D labeled, also size-mutable array

Why more than one data structure?¶

"The best way to think about the pandas data structures is as flexible containers for lower dimensional data. For example, DataFrame is a container for Series, and Panel is a container for DataFrame objects. We would like to be able to insert and remove objects from these containers in a dictionary-like fashion."

Merge¶

Pandas provides various facilities for easily combining together Series and DataFrames. Dependent on the problem, the data has to be merged in a different way:

  1. Combine objects with overlapping indeces, where df1 are considered to be of 'higher quality', e.g.:

    df = df1.combine_first(df2) # values of df1 are prefered

  2. Append objects as rows to another object, e.g.:

    df = df1.append(df2) # the dataframes are stacked on one another

  3. Concatenate objects as rows or columns, e.g.:

    df = pd.concat([df1, df2], axis=1) # columns of dataframes are combined to one dataframe

  4. ...

There are even more methods (e.g. df.join, pd.merge) to merge Pandas objects optimized for certain circumstances. But most of the time, there isn't just one way of doing it. Please refer to the respective section in the documentation for more information.

Example: Split the DataFrame object and merge the resulting DataFrames again to a single one.

In [136]:
# Slice two DataFrames
df1 = df[['Tmin', 'Tmax']]
df2 = df[['Tavg', 'dT']]

# Concatenate them again columnwise
df = pd.concat([df1, df2], axis=1)

Grouping¶

"Group by" referes to a process involving:

  • Splitting the data into groups based on some criteria (.groupby() method)
  • Applying a function to each group independently (e.g. .mean())
  • Combining the results into a data structure (done by Pandas by default - customizable)

Example: Calculate annual means of all colums in DataFrame

df_annual_means = df.groupby(by=df.index.year).mean()
